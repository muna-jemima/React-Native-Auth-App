# React Native Auth App

This is a simple React Native authentication app built with Expo. I created this project while practicing React Native mobile development, focusing on authentication screens, navigation, UI design, and project structure.

## Project Purpose

The purpose of this project was to practice building a basic mobile authentication flow using React Native.

## Features

- Onboarding screen
- Login screen
- Registration screen
- Forgot password screen
- Home screen
- Navigation between screens
- Mobile-friendly user interface
- Custom image assets

## Screens

- OnboardingScreen
- LoginScreen
- RegisterScreen
- ForgotPasswordScreen
- HomeScreen

- React Native

## How to Run the Project

Clone the repository:

```bash
git clone https://github.com/muna-jemima/React-Native-Auth-App.git
```

Go into the project folder:

```bash
cd React-Native-Auth-App
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npx expo start
```
